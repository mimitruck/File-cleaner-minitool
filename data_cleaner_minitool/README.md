# Data Cleaner GUI (PySide6 + pandas)

A small open-source desktop tool to preview and clean tabular data.

## Features
- Load CSV / Excel (.xlsx / .xls)
- Preview first N rows
- Convert dtype for a selected column (string / int / float / datetime)
- Regex-based cleaning for a selected column
- Export to CSV / XLSX

## Install

pip install -r requirements.txt
